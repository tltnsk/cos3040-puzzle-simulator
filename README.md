# Puzzle Simulator - COS3040 Project 

This is a project for the COS3040 course. The project is a puzzle simulator written in Python.

Players solve different types of puzzles such as riddles, word puzzles, and mathematical puzzles.

The system is designed using OOP principles, including inheritance, polymorphism, encapsulation, as well as operator overloading.

## Features
- Multiple puzzle types
- Three-level class hierarchy
- JSON file loading and saving
- Puzzle filtering and sorting
- Player statistics
- Unit testing

## Technologies
- Python
- JSON
